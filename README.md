## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/EmelyanovKonstantin/TheArcticData.git
   ```
2. Install Java 17
3. Run for Windows:
   ```sh
   mvnw.cmd spring-boot:run
   ```
   or of Linux:
   ```sh
   ./mvnw spring-boot:run
   ```
